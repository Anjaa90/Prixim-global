Types of Cloud Development Services

This model is very popular among the general public. Software-as-a-service provides end users with an application as a service over the Internet. The cloud service provider manages the setup and maintenance of basic computing services. However, consumers must purchase a subscription to use the service.

Platform-as-a-service (PaaS) provides developers with a hassle-free environment to develop, test, and deploy applications on demand. However, the cloud provider manages and configures every computing resource that the environment (platform) needs. Heroku, SAP Cloud, Docker, Google App Engine and Azure SQL Database provide these services.

Serverless Computing, such as PaaS, allows developers to focus on application development without managing the underlying environment. However, the most significant differences are in architecture, tools, scalability, and cost.

Infrastructure-as-a-service (IaaS) leases the building blocks—servers, storage, networks, and operating systems (OS)—of cloud IT infrastructure. A cloud provider provides organizations with a digital model of an on-premise setup. This allows organizations to scale up and down based on business demand without having to physically build or purchase infrastructure.
